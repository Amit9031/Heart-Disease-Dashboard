# ❤️ Heart Disease Analysis Dashboard using Power BI

## 📌 Project Overview

This project focuses on analyzing heart disease patient data using Power BI to identify major health risk factors and provide meaningful healthcare insights through interactive dashboards and visualizations.

The dashboard helps in understanding patient health trends, monitoring important KPIs, and identifying the key contributors to heart disease.

---

## 🎯 Objectives

* Analyze patient health records to identify heart disease risk factors.
* Build interactive dashboards for healthcare analytics.
* Visualize patient trends related to cholesterol, blood pressure, and chest pain.
* Generate actionable insights for better clinical decision-making.

---

## 📊 Key Insights

* Identified **Age Group**, **High Cholesterol**, and **Chest Pain Type** as the top risk factors.
* Analyzed 1,000+ patient records for pattern recognition and trend analysis.
* Built KPI cards and drill-down reports for detailed health monitoring.
* Improved data interpretation using interactive visuals and filters.
